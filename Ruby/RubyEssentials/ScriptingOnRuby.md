+ Good practice
    + To start with a shebang ``` #!/usr/bin/env ruby```
    + This tells the system to handle the file with whatever ruby we have installed
    + Comment codes
+ Exit a running script
    + exit, exit!
    + abort(msg)
    + con with the classic control + c
